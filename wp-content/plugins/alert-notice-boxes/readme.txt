== Description ==

Alert Notice Boxes allows you to create beautiful custom alerts that appear on pages or posts of your choice.

Demo - <a href="http://wp-plugins-demos.madadim.co.il/" target="_blank">Alert Notice Boxes - Live Preview</a>

= Features: =

* Create unlimited notification alerts boxes
* Alerts appear by device type selection - NEW
* Create custom notification designs - NEW
* Can be added shortcods to the message content, like a newsletter - NEW
* Can be added embed code, like YouTube or Facebook - NEW
* You can choose where the alert will appear on the site in general or on a specific page - NEW
* Control delay
* Control show Time
* Control the location of the alerts - NEW
* Create in and out animation for notification - NEW
* Control how often an alert appears - NEW
